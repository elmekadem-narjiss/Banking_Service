# Skypay Technical Test 1 - Banking Service

## Description générale

Ce projet implémente un service bancaire simple en Java,

Les fonctionnalités principales sont :
- Dépôt d'argent (`deposit`)
- Retrait d'argent (`withdraw`)
- Affichage du relevé de compte (`printStatement`)

Le relevé affiche les transactions dans l’ordre chronologique inverse (la plus récente en premier), avec la date, le montant et le solde cumulé.
